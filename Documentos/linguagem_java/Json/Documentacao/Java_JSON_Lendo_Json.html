Java – JSON


JSON é um recuso de transformação de objetos de qualquer tipo em um literal.

O intuito deste artigo é mostrar como fazer uma simples aplicação que tranforma objetos em literal e lê os dados de um literal.

Veja o exemplo:

view source
print?
01.package br.com.rmt.testes.json;
02. 
03.import org.json.JSONObject;
04. 
05.public class TesteJSON {
06.public TesteJSON(){
07.//this.escrever();
08.this.ler();
09.}
10. 
11.public void escrever(){
12.JSONObject jo = new JSONObject();
13.try {
14.jo.put("nome", "Ronneesley");
15.jo.put("sobrenome", "Moura Teles");
16.} catch (Exception ex){
17.System.out.println("Erro Escrita: " + ex.getMessage());
18.}
19.System.out.println(jo.toString());
20.}
21. 
22.public void ler(){
23.String json = "{\"sobrenome\":\"Moura Teles\",\"nome\":\"Ronneesley\"}";
24. 
25.try {
26.JSONObject jo = new JSONObject(json);
27. 
28.System.out.println("Nome: " + jo.get("nome"));
29.System.out.println("Sobrenome: " + jo.get("sobrenome"));
30.} catch (Exception ex) {
31.System.out.println("Erro Leitura: " + ex.getMessage());
32.}
33. 
34.}
35. 
36.public static void main(String[] args){
37.new TesteJSON();
38.}
39.}

Para executar o código é recomendado utilizar a plataforma Eclipse ou o Netbeans, que facilitam a inclusão de outros códigos fontes externos. Assim deve ser efetuado o download do arcabouço disponível no site http://www.json.org/java/index.html e adicioná-lo como dependência do projeto.
